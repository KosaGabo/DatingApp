Welcome to my Webprojekt application!

In the application, all the features are working, we have the option to register, like other users and send messages. We can also edit our profile and upload photos. Currently, the site is only optimized for PC, more platforms are under development.

On the frontend, I used Angular 14 and on the backend I used C# with the .NET7 framework. The source code is available to view on my Github page.

To create this site, I used the Udemy course 'Neil Cummings: Build an app with ASP.NET Core and Angular from scratch.'
